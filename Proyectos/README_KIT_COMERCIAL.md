## 🎯 FLUJO DE VENTA (Paso a paso)

### FASE 1: Prospección (Día 1-3)

```
1. Busca contactos de agencias ASSIST CARD
   → LinkedIn: "ASSIST CARD" + "agencia" + [estado]
   → Google Maps: "ASSIST CARD MEXICO"
   → Extrae 10-20 contactos

2. Personaliza EMAIL_PROPUESTA_TEMPLATE.md
   → Menciona algo específico de SU portal
   → No envíes masivo, personaliza cada uno

3. Envía primer mensaje (WhatsApp o LinkedIn)
   Ejemplo personalizado:
   "Vi el portal de cotizaciones de [Agencia] 
    y me pareció que podría ser mucho más rápido...
    ¿Te gustaría una idea?"

4. Haz seguimiento (Día 5, 10)
```

**Resultado esperado:** 30-50% dirán "sí, cuéntame más"

---

### FASE 2: Descubrimiento (Día 4-7)

```
1. Reciben "cuéntame más" o confirman llamada

2. Lee LLAMADA_DISCOVERY_CHECKLIST.md
   → Prepárate las 8 preguntas
   → Ten calculadora a mano
   → Toma notas

3. Haz llamada de 20-30 min
   → Escucha 70%, habla 30%
   → Calcula ROI en vivo ("esto te ahorra $10k/mes")
   → Sugiere paquete según volumen

4. Después de llamada
   → Envía resumen + presupuesto por email
```

**Resultado esperado:** 50% dirán "sí, envíame presupuesto formal"

---

### FASE 3: Propuesta (Día 8-14)

```
1. Consulta PRESUPUESTOS_CUSTOMIZADOS_EJEMPLOS.md
   → Elige escenario más parecido
   → Adapta números a SU caso

2. Personaliza ASSIST_CARD_PRESUPUESTO.md
   → Cambiar nombres/números
   → Agregar elementos custom si aplica

3. Envía presupuesto formal
   → Validez: 30 días
   → Incluye contrato draft
   → Incluye timeline

4. Haz un follow-up
   → "¿Tienes preguntas sobre la propuesta?"
```

**Resultado esperado:** 40-60% dirán "sí, vamos"

---

### FASE 4: Contratación (Día 15-20)

```
1. Reciben confirmación

2. Envía contrato firmado + factura
   → 50% al iniciar
   → 50% a la entrega

3. Negocias acceso a:
   → Sistema actual (si aplica)
   → Datos de clientes (anónimos)
   → APIs (si necesitas integrar)
   → Credenciales de n8n/hosting

4. Comenzar desarrollo (Día 20)
```
