Un programa de Organizacion en Python para ingresos gastos y Total separando:
Lineas de credito del monto actual tras restar pagos a tdc o similar
Pero con un Sistema de relaciones entre compras o gastos y TDC (por ejemplo: $500 gastado en: TDC)
Incluyendo Tambien fechas de corte, fechas limites de pago y fecha actual y permita anexar pdf o comprobantes a cada transaccion
